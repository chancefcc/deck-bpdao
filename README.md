# Breakpoints.live Deck on [Slidev](https://github.com/slidevjs/slidev)!

## Local Writing

Git clone to local dir, and start the slide show:

- `yarn install`
- `yarn dev`
- visit <http://localhost:3030>

Edit the [slides.md](./slides.md) to see the changes.

## Deploy

This repo is hosted at https://github.com/chancefcc/deck-bpdao, and auto deployed to Cloudflare Pages by running,

```
git push origin main
```

Once you finish above, you'll see the live deck at [https://deck-bzb.pages.dev/](https://deck-bzb.pages.dev/).

Learn more about Slidev at the [documentation](https://sli.dev/).
